# Survey Analyzer (Python)

A beginner Python project that collects survey data, analyses it, and exports results to a CSV file.

---

## Project Overview

This program collects multiple survey responses from users and performs basic data analysis including:

* Calculating average age
* Calculating average user rating
* Identifying the most common favourite app
* Displaying all responses
* Saving data to a CSV file

---

## Features

* Collect multiple survey responses using user input
* Store structured data using dictionaries and lists
* Input validation for user choices
* Calculate averages
* Frequency analysis (most common app)
* Export results to CSV

---

## Concepts Practised

* Functions
* Loops (while and for)
* Dictionaries
* Lists
* Input validation
* Error handling
* Data aggregation
* File handling (CSV)
* Basic data analysis logic

---

## Example Output

### User Input

* Age: 25
* Favourite app: Instagram
* Hours online: 3.5
* Satisfaction rating: 8

### Analysis Output

* Average age: 26.3
* Average satisfaction rating: 7.8
* Most common app: Instagram

survey_results.csv
